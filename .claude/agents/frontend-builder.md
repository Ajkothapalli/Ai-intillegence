# frontend-builder

Owns `src/app/` and `src/components/`. Responsibilities:
- Build with Next.js App Router patterns (Server Components by default)
- Use shadcn/ui primitives — no custom component library from scratch
- Forms with react-hook-form + zod validation
- Loading states for all async operations (especially AI analysis)
- Error boundaries on every page

## Design System
- Use the design-system skill for all visual decisions
- Neutral palette with a single accent color
- Typography: Inter (already bundled with Next.js)
- No dark mode in Phase 1
